<summary><b>E11</b></summary>

Como $|z_k|=1$, tenemos $z_k\bar{z_k}=1$, o sea $\bar{z_k} = 1/z_k$.

$|z_1-z_2|^2 = (z_1-z_2)(\bar{z_1}-\bar{z_2}) = 2 - z_1/z_2 - z_2/z_1$

De $z_1+z_2+z_3=0$: $z_3 = -(z_1+z_2)$. Como $|z_3|=1$:
$|z_1+z_2|^2 = 1 \implies 2 + z_1/z_2 + z_2/z_1 = 1 \implies z_1/z_2 + z_2/z_1 = -1$

Entonces $|z_1-z_2|^2 = 2 - (-1) = 3$, así que $|z_1-z_2| = \sqrt{3}$.

Por simetría, los tres lados miden $\sqrt{3}$ → equilátero.

<summary><b>E13</b></summary>

$\left|\frac{z-a}{1-\bar{a}z}\right|^2 = \frac{(z-a)(\bar{z}-\bar{a})}{(1-\bar{a}z)(1-a\bar{z})}$

Como $|z|=1$, $z\bar{z}=1$. El numerador: $1 - z\bar{a} - a\bar{z} + a\bar{a}$.

¡Son iguales! Así que el cociente es $1$, y el módulo también.
